# 🌍 Multilingual Smart Calculator (多機能・多言語対応電卓)

## 📖 概要 (Overview)
   クリック音の有無を切り替え可能です。(sound effect mood)
---

## ✨ 主な機能 (Key Features)

### 1. 🔊 サウンド機能 (Sound Effects)
Sound ON/OFF: クリック音の有無を切り替え可能。操作時のフィードバックを重視。

### 2. 🔢 基本計算 (Basic Arithmetic)
加算、減算、乗算、除算、パーセント計算など、日常的な計算を網羅。

---

## 🛠️ 使用技術 (Tech Stack)
* Language: C#
* Framework: .NET (WPF/Windows Forms)
* IDE:Visual Studio
* Design: Custom UI Components

* 拡張性: オブジェクト指向 (OOP) を意識し、将来的に他の言語や機能を追加しやすいコード設計にしています。
